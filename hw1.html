Directions

This text Adventure Game deals with moving around the grid looking for the exit door and finding either prizes (W) or monsters (C) to fight. You start off with 100 points and either goes down if you lose to a monster or goes up if you find prizes. The Objective is to find the exit key. Player loses if score reaches zero. The game also can end if the player is able to find the exit.

Score : 100pts
